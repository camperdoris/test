安裝

要使用gitbook，要先安裝 Node.js 並通過 npm 來安裝 

npm install gitbook-cli -g




指令

1、初始化書藉目錄 gitbook init
2、編譯和執行 gitbook serve
3、開啟本地4000端口 http://localhost:4000/



改變電子書目錄結構

1、在桌面新增一個資料夾test
2、使用編輯器開啟test文件夾
3、SUMMARY.md 這個是存放書的目錄結構
4、book.json 這個是放入插件的json檔

5、插件

Github Button插件

通過 npm 來安裝 

npm install gitbook-plugin-github-buttons


將二段程式碼分別加入book.json中並存檔 

{
  "plugins": [
    "github-buttons"
  ],
  "pluginsConfig": {
    "github-buttons": {
      "buttons": [{
        "user": "azu",
        "repo": "JavaScript-Plugin-Architecture",
        "type": "star",
        "size": "small"
        }]
    }
  }
}


gitbook install-只要有修改到json檔都要重新安裝

gitbook serve
